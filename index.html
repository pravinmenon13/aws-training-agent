      Certification Renewals
      Always keep your credentials relevant and active — L&K provides FREE vouchers for all renewal levels
      🌳 Foundation — AWS Cloud Practitioner (Free, No Exam!)

      Renew via Cloud Quest Recertify Game on AWS Skill Builder

      No voucher, no Skill Builder subscription, no exam required!
      1. Log into AWS Skill Builder using the same account used for your cert (IBM ID / SSO)
      2. Search for Cloud Quest Recertify and enrol
      3. Change role to Recertify (not Cloud Practitioner mode)
      4. Enter your AWS ID linked to your certification
      5. Complete all 13 levels → certification extended by 3 years
      6. Claim the certification extension on completion

      ⚠ Cert must be active & expiry within 6 months. If expired: raise LRT request as renewal.

      🎍 Associate, Professional & Specialty — Free Voucher via LRT

      Steps to get your renewal voucher from L&K

      Applies to all Associate, Professional and Specialty level certifications
      1. Log into LRT
      2. Select your certification for renewing from the catalogue
      3. Mark as Renewal for correct tracking
      4. Set a tentative exam date (approx. 1 week ahead)
      5. Submit with justification — "cert renewal" is sufficient
      6. Voucher allocated within 7–10 business days depending on availability
